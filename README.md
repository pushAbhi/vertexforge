# VertexForge - Company Landing Page

Simple responsive company website.
Built with Next.js 16 + Tailwind + TypeScript. Features Google reCAPTCHA and Resend email delivery.

**Live Demo:** [vertexforge-alpha.vercel.app](https://vertexforge-alpha.vercel.app)

## Tech Stack
- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Google reCAPTCHA**
- **Resend** (for emails)

## Installation

## 1. Clone the repo
```bash
git clone https://github.com/pushAbhi/vertexforge.git
cd vertexforge
```

## 2. Install dependencies
```bash
npm install
```

## 3. Setup environment variables
```bash
cp .env.example .env
```

## 4. Launch
```bash
npm run dev
```
